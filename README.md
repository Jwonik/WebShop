# WebShop
웹을 이용한 온라인 쇼핑몰 프로젝트입니다. </br>

# Description
+ 개발 기간: 2022.11.29 ~ 2022.12.27

+ 참여 인원: 5명

+ 사용 기술
  + Java, HTML, CSS, JavaScript
  + Ajax, JQuery
  + Oracle 18c
  
+ 담당 구현 파트
  + Oracle DB 연결 및 테이블 설계
  + 회원가입, 수정, 탈퇴
  + API를 통한 결제
  + 주문, 주문 내역, 주문 상세 내역 관리
  + 관리자 기능 회원, 주문, 환불 관리

# 후기
팀 프로젝트를 진행하는 데 있어 가장 중요한 요소는 의사소통이라는 것을 다시 되새겼다. </br>
사전에 변수명, 클래스명 등 규칙을 정하고 프로젝트를 진행하지 않아 최종적으로 프로젝트를 합치는 데 어려움을 겪었다. </br>
